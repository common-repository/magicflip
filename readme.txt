MagicFlip - simple PDF viewer plugin for WordPress

== Description ==

MagicFlip is a plugin that enables you add PDF documents & books to your WordPress website.


== How To Use ===
To insert a book into a web page, you should use the shortcode "magicflip", for example [magicflip src="mysite.com/book.pdf"].
Shortcode parameters:
src - pdf document url
class - additional css classes applied to the map container
width - map container width, the value can be in px or %
height - map container height, the value can be in px or %

== Installation ==

* go to the Wordpress admin console
* go to "Plugins" -> "Add New"
* search for "MagicFlip"
* Click "Install Now"
* Click "Activate"
